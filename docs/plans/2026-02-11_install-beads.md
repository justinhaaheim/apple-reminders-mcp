## Design Decisions

- **Beads supplements, does not replace** the scratchpad/ROADMAP system. Beads is for concrete tasks with status tracking; scratchpads are for design notes and session context.
- **SessionStart hook only runs on Claude Code Web** (`CLAUDE_CODE_REMOTE=true`). Local users install `bd` manually.
- The hook also runs `bun i` to ensure deps are installed in web environments.
